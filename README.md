# Drawing Tool

## feature

- 드로잉 타입 선택: 직선, 곡선, 원, 직사각형, 다각형
- 선 두께 선택
- 컬러 선택
- Undo, Redo
- - 새로 고침 이후에도 캔버스 내용 유지

### 개발 서버 실행

```bash
yarn
yarn dev
```
